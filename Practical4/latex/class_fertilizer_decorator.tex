\doxysubsection{Detailed Description}
A concrete decorator class that adds fertilizer-\/related functionality to a \doxylink{class_farm_unit}{Farm\+Unit}. 

The \doxylink{class_fertilizer_decorator}{Fertilizer\+Decorator} class enhances the functionality of a \doxylink{class_farm_unit}{Farm\+Unit} by adding features related to fertilizers, such as improving crop yield, harvesting crops, and calculating the remaining storage capacity. It follows the \doxylink{class_decorator}{Decorator} design pattern to dynamically add these functionalities.
