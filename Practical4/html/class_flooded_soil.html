Detailed Description

A concrete state representing flooded soil in the State design pattern.

FloodedSoil is one of the possible states of soil within the State design pattern. This state indicates that the soil is flooded, and it defines behaviors such as harvesting crops and reacting to further rain. When rain occurs, the state of the soil may change.
